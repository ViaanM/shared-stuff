5. Nests


Nests don't change anything in our website. They make sure everything is working.
The nest !DOCTYPE html tells the computer that this is a html file.
The nest html tells the computer that everything in the middle is html.
The nest head should contain all code that does not impact the website's looking.
The nest body should contain all code that impacts the website's looking.
